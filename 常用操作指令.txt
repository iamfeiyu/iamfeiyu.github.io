### 常识
.gitignore 该文件描述的是git忽略推送的目录及文件

### 查看依赖包的版本
将展示旧版本、合适的版本（Wanted）、最新版本
npm outdated

### 更新依赖包到合适的版本
更新依赖包的版本到Wanted（合适的版本）。能够解决，本来正常，一段时间后突然不正常的情况。比如，部署总是失败。
npm update

### new
新建一篇文章。如果没有设置layout的话，默认使用_config.yml 中的 default_layout 参数代替。如果标题包含空格的话，请使用引号括起来。
hexo new [layout] <title>

### generate
生成静态文件。
hexo generate

# 简写
hexo g


### server
启动本地服务器实现预览，访问地址为：http://localhost:4000/。

hexo server
# 简写
hexo s


### deploy
部署到Github
hexo deploy

# 简写
hexo d


### 常用指令
 
# 标准参数启动服务器，生成和部署
hexo clean && hexo g && hexo d


# 标准参数启动服务器，生成和本地预览
hexo clean && hexo g && hexo s
 

# 生成静态文件后部署 
hexo g -d
